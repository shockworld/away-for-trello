Away for Trello

Support

Away for Trello is a free, open-source Power-Up maintained by Shockworld.

Quick setup

  1. Add the Power-Up to a Trello board.
  2. Open Power-Ups → Away for Trello → Settings.
  3. Enter the person, status, dates, and message.
  4. Enable Publish this notice and save.

The notice is not visible

  • Confirm that board-buttons, card-badges, and show-settings are enabled in Trello's app administration portal.
  • Refresh the board after changing Power-Up capabilities.
  • On narrow screens Trello may collapse the board button, but the colored notice remains visible on cards.
  • If the return date has arrived, the old notice intentionally expires.
  • Read-only visitors only see the button while a notice is active.

Report a problem

Open an issue in the GitHub repository. Include the browser, steps to reproduce, and expected behavior, but never include private board content, credentials, or personal information.
